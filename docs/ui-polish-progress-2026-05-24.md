# UI 수정 진행상황 (2026-05-24)

## 완료

- 2차 수정: `/upload`의 E2 표면 선택 아래에 결과 카드와 E2 대시보드가 바로 이어지도록 배치.
- 2차 수정: E2에서는 자동 파싱 상태 카드가 표면 선택과 결과 카드 사이에 끼지 않도록 정리.
- 2차 수정: 상단 active 내비게이션 버튼의 검정 배경에서도 글씨가 보이도록 흰색 텍스트를 강제 적용.
- 2차 수정: `/upload` 시나리오 선택 카드를 데스크톱 기준 5열 그리드로 조정해 E1~E5가 한 줄에 나란히 보이도록 수정.
- 2차 수정: 홈의 `분석 화면` 카드에서 `CSV Upload`를 제거하고 `Scenario Dashboard -> Ablation -> Results -> Method` 4개 카드 구성으로 변경.
- 2차 수정: 상단 내비게이션도 `Scenario Dashboard -> Ablation -> Results -> Method` 순서로 정리.
- 3차 수정: 그래프 알고리즘 팔레트를 Clean Lab Scientific으로 확정.
- 3차 수정: `Raw ToF #6B7280`, `Fixed KF #2563EB`, `CM-AKF #16A34A`, `TinyML-AKF #7C3AED`로 차트/범례/알고리즘 토글 색상 통일.
- 4차 수정: E2 표면 선택 아래 결과 카드도 알고리즘 팔레트로 통일. Raw는 회색, Fixed는 블루, CM은 그린, TinyML은 보라로 표시.
- 4차 수정: Ablation 표 5-3의 Fixed/CM/TinyML 컬럼을 알고리즘 팔레트로 정리하고, 기존 레드 강조였던 `3f 모델 폭발`, 별표 수치, 양수 열화값은 레드로 복원.
- 4차 수정: Results 페이지에 RQ1 블루, RQ2 그린, RQ3 보라 톤의 요약 카드와 표 컬럼 색상을 적용.
- 4차 수정: Method 페이지에 CSV Parse, Metrics, TinyML, Limit 분류 카드와 섹션별 색상 강조를 적용.
- 홈 화면 제목 줄바꿈과 여백을 조정해 `STM32 센서 데이터 파이프라인 · 실시간 시각화 풀스택 시스템` 문구가 잘리지 않도록 수정.
- 홈 화면의 기술 플로우, 작은 기술 배지, `바로 시작하기 / 데모 데이터 로드` 박스를 삭제해 첫 화면 정보량을 줄임.
- 홈 화면 핵심 지표 4개 카드를 회색 박스 기반으로 통일하고, 숫자 외 설명 글자 크기를 키움.
- 상단 좌측 타이틀을 `Edge AI Kalman Dashboard`만 남기고 `졸업연구` 문구를 삭제.
- 타이틀 클릭 시 `/` 홈 화면으로 돌아가도록 변경.
- 상단 내비게이션을 영어로 통일하고 `Ablation` 버튼을 추가.
- `/upload`에서 `데이터 불러오기` 버튼 없이 시나리오 선택 즉시 CSV 자동 파싱 및 대시보드 표시.
- `/upload`에서 `고급: 직접 CSV 업로드` 섹션 삭제.
- E1, E2, E3, E5 자동 로드 후 인라인 대시보드가 바로 보이도록 변경.
- E4 선택 시 별도 대시보드 버튼 없이 정적 장기 안정성 대시보드 즉시 표시.
- 런 선택 기본값을 `All (평균)`으로 변경하고, 런 탭에서 `All (평균)`을 첫 번째에 배치.
- 차트 1 범례 순서를 `Raw ToF -> Fixed KF -> CM-AKF -> TinyML-AKF`로 고정.
- E2 흰/검정/투명 표면 선택 시 같은 형식의 회색 성능 카드가 유지되고 숫자만 바뀌도록 통일.
- E2 `반사특성 비교` 제목과 주요 문구 크기 확대.
- E3 `알고리즘별 성능` 표 제목, 배지, 셀 글자 크기와 좌우 여백 조정.
- Results 페이지의 `RQ1`, `RQ2`, `RQ3` 강조 색을 차콜 계열 하나로 통일하고 라벨 크기를 확대.
- 전체 강조색을 블랙/화이트/그레이 중심으로 정리.
